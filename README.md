## 🏗 Project Overview

Haler Official Website.

## Deployment
- **Official Live Site**: [https://haler.vercel.app](https://haler.vercel.app)
- **Platform**: Vercel

## 📁 Key Directories
- `/app`: Next.js App Router (Pages & API)
- `/components`: Reusable UI Components
- `/workbench`: **Safe Development Area** (Drafts & Refactoring)
- `/assets`: Raw Image & Video Sources
- `/backups`: Production Sync & Legacy Archives
- `/kickstarter`: Historical Campaign Records

## 🚀 Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS / Vanilla CSS
- **Database**: Supabase
- **E-commerce**: Shopify Admin/Storefront API
- **Deployment**: Vercel

## Key Features (Mobile PWA)
- **Everyday Threats Section**: 500vh sticky scroll sequence with triple-row infinite carousel and snappy popup animations.
- **Anatomy Section**: Interactive respiratory hydration guide.
- **Bliz Intro**: 3D-like product reveal.
